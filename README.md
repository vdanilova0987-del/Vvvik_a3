Данная программа реализует сервер именованного канала (Named Pipe) с именем mypipe. После запуска сервер ожидает подключения клиента. Когда клиент подключается и отправляет текстовое сообщение, программа принимает его, сохраняет в файл log.txt вместе с текущими датой и временем, затем преобразует полученный текст в верхний регистр и отправляет результат обратно клиенту.

Программа написана на языке C# с использованием платформы .NET. Для обмена данными применяется технология Named Pipes, позволяющая организовать взаимодействие между процессами на одном компьютере.

Для запуска программы необходимо выполнить команду dotnet run в каталоге проекта.
